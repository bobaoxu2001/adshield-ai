# Project Brief

## Product

AdShield AI is a decision-support workspace for commercial ads risk strategy and operations. It exposes the full loop from public-source evidence to a reviewable recommendation and system-level operating metrics.

## Target user

A risk strategy analyst, policy operations analyst, or bilingual reviewer managing high-volume commercial content without sacrificing explainability or escalation controls.

## Core loop

Ingest real public records → normalize and validate → extract bilingual signals → retrieve summarized policies → score and route → collect reviewer feedback → diagnose coverage and quality.

## Success criteria

- Works without paid API keys.
- Never shows fabricated production metrics or examples.
- Keeps source provenance and limitations visible.
- Supports a real reviewer decision write-back.
- Explains how each metric maps to operational decisions.
- Keeps complaint-derived risk priors structurally separate from ad-enforcement actions.
- Makes optional LLM comparison explicit, on-demand, schema-constrained, and failure-isolated.
